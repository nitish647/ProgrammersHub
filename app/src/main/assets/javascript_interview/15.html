Answer:Currying is the process of taking a function with multiple arguments and turning it into a sequence of functions each with only a single argument. Currying is named after a mathematician Haskell Curry. By applying currying, a n-ary function turns it into a unary function. Let's take an example of n-ary function and how it turns into a currying function

const multiArgFunction = (a, b, c) => a + b + c;

const curryUnaryFunction = a => b => c => a + b + c;

curryUnaryFunction (1); // returns a function: b => c =>  1 + b + c

curryUnaryFunction (1) (2); // returns a function: c => 3 + c

curryUnaryFunction (1) (2) (3); // returns the number 6

Curried functions are great to improve code re-usability and functional composition.
