Tuple

A tuple is similar to the list in many ways. Like lists, tuples also contain the collection of the items of different data types. The items of the tuple are separated with a comma (,) and enclosed in parentheses (). A tuple is a read-only data structure as we can't modify the size and value of the items of a tuple.

Adding Tuples

Tuple can be added by using the concatenation operator(+) to join two tuples.

Example

Output

tuple = ( 'abc', 123 )

tuple1 = ('hi', 10.12 , 10) print tuple+tuple1

('abc', 123, 'hi', 10.12, 10)


Replicating Tuple

Replicating means repeating. It can be performed by using '*' operator by a specific number of time.

Example

Output

tuple = ( 'abc', 123 )

print tuple*2

('abc', 123, 'abc', 123)


Tuple Slicing

A subpart of a tuple can be retrieved on the basis of index. This subpart is known as tuple slice.

Example

Output

print tuple[0] print tuple[2:4] print tuple[3:]

print tuple[-3:-1]

Tuple Deleting

Deleting individual element from a tuple is not supported. However the whole of the tuple can be deleted using the del statement.

Example

Output

tuple = ( 'abc', 123 , 'hi', 10.12 , 10 ) print tuple

del tuple

print tuple

min(tuple) Method

This method is used to get min value from the sequence of tuple.

Example

Output

tuple = ( 'abc', 123 , 'hi', 10.12 , 10 )

print min(tuple)

10


max(tuple) Method

This method is used to get max value from the sequence of tuple.

Example

Output

tuple = ( 123 , 10.12 , 10 )

print max(tuple)

This method is used to get length of the tuple.


Example
