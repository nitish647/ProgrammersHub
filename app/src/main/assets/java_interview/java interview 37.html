Answer: We use the join() method for joining one thread with the end of the

currently running thread. It is a non-static method and has an overloaded version. Consider the example below:

public static void main (String[] args) {

Thread t = new Thread();

t.start();

t.join();

}

The main thread starts execution in the aforementioned example. As soon as the

execution reaches the code t.start() then the thread t starts its own stack for

execution. The JVM switches between the main thread and the thread there. Once

the execution reaches the t.join() then the thread t alone is executed and allowed to

complete its task. Afterward, the main thread resumes execution.
