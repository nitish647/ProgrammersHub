Answer: A Singleton class allows only one instance of the class to be created. A class can be made singleton with the following steps:

Creating a static instance of the class with the class. By not allowing the user to create an instance with default constructor by defining

private constructor. Create a static method to return the object of an instance of A. public class Singleton

{

public static void main(String args[])

{

Single obj1 = Single.getInstance(); /* both would point to one and same instance

of the class */

Single obj2 = Single.getInstance();

}

}

class Single

{

static Single obj = new Single(); /* step a*/

private Single() /* step b*/

{

}

public static Single getInstance()

{

return obj; /* step c*/

}
