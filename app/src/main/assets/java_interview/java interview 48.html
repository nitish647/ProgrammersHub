Boxing: The concept of putting a primitive value inside an object is called boxing. Unboxing: Getting the primitive value from the object. Autoboxing: Assigning a value directly to an integer object. Auto unboxing: Getting the primitive value directly into the integer object. public class BoxUnbox

{

public static void main(String args[])

{

int i = 5;

Integer ii = new Integer(i); /*Boxing*/

Integer jj = i; /*Unboxing*/

int j = jj.intValue(); /*Unboxing*/

int k = jj; /*AutoUnboxing*/

}
