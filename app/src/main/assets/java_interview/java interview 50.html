Answer: Different types of typecasting are:

Implicit: Storing values from a smaller data type to the larger data type. It is

automatically done by the compiler. Explicit: Storing the value of a larger data type into a smaller data type. This results

in information loss:

Truncation

Out of Range

Let us see code samples to further understand this:

int i = 10;

long l = i;

long l = 10,000;

int i = (int) l;

float f = 3.14f

int i = (int) f;

i=3;

long l = 123456789;

byte b = (byte) l;
