Answer: A Java Map is an object that maps keys to values. It can’t contain

duplicate keys and each key can map to only one value. In order to determine

whether two keys are the same or distinct, Map makes use of the equals() method. There are 4 types of Map in Java, described as follows:

HashMap - It is an unordered and unsorted map and hence, is a good choice when

there is no emphasis on the order. A HashMap allows one null key and multiple null

values and doesn’t maintain any insertion order. HashTable – Doesn’t allow anything null and has methods that are synchronized. As

it allows for thread safety, the performance is slow. LinkedHashMap – Slower than a HashMap but maintains insertion order and has a

faster iteration. TreeMap – A sorted Map providing support for constructing a sort order using a

constructor.
